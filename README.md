# [Chit Chat Server](https://chitchat-server.now.sh)


#### Stack

- Http Server
- Socket.io


#### Quick Start

```shell
$ git clone git@github.com:aziz-ka/chitchat-server.git
$ cd chitchat-server
$ yarn
$ yarn start
```


#### Yarn/NPM Commands

|Script|Description|
|---|---|
|`yarn start`|Start the app on `localhost:3001`|
